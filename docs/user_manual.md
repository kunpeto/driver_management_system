## 系統登入

### 登入步驟

1. 開啟瀏覽器，前往系統網址
2. 輸入帳號和密碼
3. 點擊「登入」按鈕

### 忘記密碼

請聯繫系統管理員重設密碼。

### 變更密碼

1. 點擊右上角使用者名稱
2. 選擇「變更密碼」
3. 輸入舊密碼和新密碼
4. 點擊「確認」

---

## 儀表板

登入後會看到儀表板，顯示：

- **員工統計**：在職/離職人數、部門分布
- **未結案提醒**：待處理的履歷數量
- **近期事件**：最近建立的履歷記錄
- **系統狀態**：Google API 連線狀態

## 員工管理

### 查看員工列表

1. 點擊左側選單「員工管理」
2. 使用篩選條件：
   - **部門**：選擇淡海或安坑
   - **狀態**：在職/離職
   - **關鍵字**：搜尋姓名或員工編號

### 新增員工

1. 點擊「新增員工」按鈕
2. 填寫必要資訊：
   - 員工編號（格式：YYMMX0XXX）
   - 姓名
   - 部門
   - 到職日期
3. 點擊「儲存」

### 編輯員工資料

1. 在員工列表中點擊員工名稱
2. 修改需要更新的欄位
3. 點擊「儲存」

### 員工調動

1. 在員工詳情頁點擊「調動」
2. 選擇目標部門
3. 選擇調動日期
4. 填寫調動原因（選填）
5. 點擊「確認調動」

### 批次匯入員工

1. 點擊「匯入」按鈕
2. 下載範本 Excel 檔案
3. 填寫員工資料
4. 上傳填好的 Excel
5. 確認匯入結果

### 匯出員工資料

1. 設定篩選條件
2. 點擊「匯出」按鈕
3. 選擇格式（Excel 或 CSV）
4. 下載檔案

## 履歷管理

### 查看履歷列表

1. 點擊左側選單「履歷管理」
2. 使用篩選條件：
   - **部門**
   - **類型**：基本/事件調查/人員訪談/矯正措施/考核通知
   - **狀態**：待處理/已轉換/已完成
   - **日期區間**

### 建立基本履歷

1. 點擊「新增履歷」
2. 填寫事件資訊：
   - 選擇員工
   - 事件日期和時間
   - 地點
   - 車號（選填）
   - 事件描述
3. 點擊「儲存」

### 轉換履歷類型

基本履歷可轉換為其他類型：

1. 在履歷詳情頁點擊「轉換類型」
2. 選擇目標類型
3. 填寫對應的詳細資訊
4. 點擊「確認轉換」

#### 事件調查表

需填寫：
- 事故時間與地點
- 證人資訊
- 原因分析
- 改善建議
- 是否歸責、責任比例（選填）

#### 人員訪談表

系統會自動帶入：
- 到職日期
- 事件前後班別資訊

需填寫：
- 訪談內容
- 訪談人員
- 訪談日期
- 訪談結果勾選
- 後續行動勾選
- 結論

#### 矯正措施表

需填寫：
- 事件概述
- 矯正行動
- 負責人
- 完成期限
- 完成狀態

#### 考核通知表

需填寫：
- 考核類型（加分/扣分）
- 考核項目
- 分數
- 核發日期
- 核准人

### 產生 Word 文件

1. 在已轉換的履歷詳情頁
2. 點擊「產生文件」
3. 系統自動產生並下載 Word 檔案
4. 檔案包含條碼供歸檔掃描

### 上傳 PDF（完成履歷）

1. 列印並簽核 Word 文件
2. 掃描為 PDF
3. 在履歷詳情頁點擊「上傳 PDF」
4. 選擇 PDF 檔案
5. 系統自動上傳至 Google Drive
6. 履歷狀態更新為「已完成」

## 駕駛競賽

### 查看排名

1. 點擊左側選單「駕駛競賽」
2. 選擇年度和季度
3. 查看排名列表

### 排名資訊

每位員工顯示：
- 排名
- 姓名
- 部門
- 累計時數
- 資格狀態
- 積分
- 獎金金額

### 資格條件

- 當季無 R/S 類違規
- 累計時數達門檻
- 非離職員工

### 獎金階層

| 排名 | 獎金 |
|-----|------|
| 第 1 名 | 5,000 元 |
| 第 2-3 名 | 3,000 元 |
| 第 4-10 名 | 2,000 元 |
| 第 11-20 名 | 1,000 元 |

### 查看個人歷史

點擊員工名稱可查看該員工的歷年競賽記錄。

## 考核管理

### 考核標準

系統預設 61 項考核標準，分類如下：

| 類別 | 說明 | 範例 |
|-----|------|------|
| D | 行為紀律 | D01 遲到 |
| W | 工作態度 | W01 服裝儀容不整 |
| O | 其他 | O01 未完成交接 |
| S | 安全 | S01 違反安全規定 |
| R | 行車事件 | R02-R05 人為疏失 |
| +M | 月度獎勵 | +M01 全勤、+M02 行車零違規 |
| +A | 差勤加分 | +A01 R班出勤 |

### 建立考核記錄

1. 點擊「考核管理」→「新增記錄」
2. 選擇員工
3. 選擇考核項目
4. 填寫事件日期和說明
5. 若為 R02-R05，需填寫責任判定查核表
6. 點擊「儲存」

### R02-R05 責任判定

R02-R05 類別需要填寫責任判定查核表：

1. 輸入時間節點（T0-T4）
2. 系統自動計算延誤秒數
3. 勾選 9 項疏失查核項目
4. 系統自動計算：
   - 疏失項數
   - 責任程度（完全/主要/次要）
   - 責任係數
   - 最終扣分

### 累計加重機制

同類別違規會累計加重：

| 次數 | 倍率 |
|-----|------|
| 第 1 次 | 1.0 倍 |
| 第 2 次 | 1.5 倍 |
| 第 3 次 | 2.0 倍 |
| 第 4 次以上 | 2.5 倍 |

### 查看員工考核摘要

1. 點擊員工名稱
2. 查看：
   - 當前分數（起始 80 分）
   - 各類別累計次數
   - 月度獎勵統計
   - 考核記錄列表

### 月度獎勵計算

每月底由管理員執行：

1. 點擊「月度獎勵」→「計算」
2. 選擇年月
3. 點擊「預覽」確認結果
4. 點擊「執行」完成計算

獎勵類型：
- **+M01 全勤**：當月無請假（+3 分）
- **+M02 行車零違規**：當月 R+S 類無扣分（+1 分）
- **+M03 全項目零違規**：當月無任何扣分（+2 分）

### 年度重置

每年 1 月 1 日由管理員執行：

1. 點擊「年度重置」
2. 確認影響範圍
3. 輸入確認碼
4. 執行重置

重置內容：
- 員工分數重置為 80 分
- 累計次數歸零
- 歷史記錄保留

## 差勤加分

### 執行差勤加分處理

1. 點擊左側選單「差勤加分」
2. 選擇年月和部門
3. 點擊「預覽」查看將建立的記錄
4. 確認後點擊「執行」

### 加分類型

| 代碼 | 說明 | 分數 |
|-----|------|------|
| +A01 | R班出勤 | +3 分 |
| +A02 | 國定假日出勤 | +1 分 |
| +A03 | 延長工時 1 小時 | +0.5 分 |
| +A04 | 延長工時 2 小時 | +1 分 |
| +A05 | 延長工時 3 小時 | +1.5 分 |
| +A06 | 延長工時 4 小時 | +2 分 |

### 處理結果

執行後顯示統計：
- 全勤人數
- R班出勤筆數
- 國定假日出勤筆數
- 延長工時筆數
- 跳過的重複筆數

## 系統設定

### Google 憑證設定

**管理員專用**

1. 點擊「系統設定」→「Google 憑證」
2. 上傳 Service Account JSON（用於 Sheets）
3. 完成 OAuth 授權（用於 Drive）
4. 測試連線

### 使用者管理

**管理員專用**

1. 點擊「系統設定」→「使用者管理」
2. 新增/編輯/刪除使用者
3. 設定角色和部門

### 考核標準管理

**管理員專用**

1. 點擊「系統設定」→「考核標準」
2. 查看/編輯考核標準
3. 匯入 Excel 更新標準

### 勤務標準時間

**管理員專用**

1. 點擊「系統設定」→「勤務標準時間」
2. 管理各勤務代碼的標準時間
3. 用於駕駛競賽時數計算

## 常見問題

### Q: 無法登入系統？

A: 請確認：
1. 帳號密碼是否正確（注意大小寫）
2. 帳號是否已啟用
3. 網路連線是否正常

### Q: 找不到某位員工？

A: 請檢查：
1. 是否選擇正確的部門篩選
2. 員工是否已離職（勾選「包含離職」）
3. 搜尋關鍵字是否正確

### Q: 無法產生 Word 文件？

A: 請確認：
1. 履歷已完成類型轉換
2. 網路連線正常
3. 若持續失敗，請聯繫系統管理員

### Q: PDF 上傳失敗？

A: 請確認：
1. 本機桌面應用是否已啟動
2. Google Drive 授權是否有效
3. 檔案大小是否在限制內（10MB）

### Q: 考核分數計算錯誤？

A: 考核分數由系統自動計算，包含：
- 基礎分數
- 責任係數（R02-R05）
- 累計加重倍率

如有疑問，請查看考核記錄的計算明細。

### Q: 班表資料不正確？

A: 班表資料每日自動從 Google Sheets 同步。如資料有誤，請：
1. 確認 Google Sheets 原始資料
2. 等待下次自動同步
3. 或請管理員手動觸發同步

### Q: 如何取消已建立的考核記錄？

A:
1. 找到該考核記錄
2. 點擊「刪除」（軟刪除）
3. 系統會自動重新計算員工分數
4. 如需還原，可點擊「還原」
